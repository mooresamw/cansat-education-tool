# Microservices

### Overview:
This directory will house all the code for the various services to power the app following a 
micro-service architecture to help take the processing load off of the Flask server.

I'm thinking that there will be a mix of code that will be run as needed / on demand as well as some
Google Cloud functions that run in the background.

### Installation:

None needed. Will either be handled by GCP, or by the Flask server.
